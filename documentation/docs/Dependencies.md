## Dependencies
```
- Node v14.15.1 (LTS) or a more recent version. While older versions may function, it is recommended to use the latest LTS version.

- npm 6.14.8 (LTS) or a more recent version. While Yarn can be used, it has not been specifically tested for this project.

- AWS CLI v2 is required, although v1 may function it has not been validated for this project.

- AWS Elastic Beanstalk CLI

- AWS RDS database running PostgreSQL.

- AWS S3 bucket for Frontend assets.

- AWS Elastic Beanstalk for Backend deployment.
```
